# Examples

These examples show the unified `node-orm-migration-guard` package and its subpath imports.

| File | Shows |
| --- | --- |
| `unified-core-sql-check.js` | Direct SQL checking through `/core` |
| `unified-drizzle-directory.js` | Drizzle generated migration directory checks |
| `unified-drizzle-runtime.js` | Drizzle database runtime wrapper |
| `unified-prisma-directory.js` | Prisma migration directory checks |
| `unified-prisma-reporter.js` | Custom CI reporting for Prisma migrations |
| `unified-knex-migration.js` | Knex migration wrapper |
| `unified-objection-migration.js` | Objection.js alias through the Knex wrapper |
| `unified-sequelize-migration.js` | Sequelize `QueryInterface` wrapper |
| `unified-typeorm-query-runner.js` | TypeORM `QueryRunner` wrapper |
| `unified-typeorm-migration-instance.js` | TypeORM migration instance wrapper |
| `unified-mikro-orm-migration.js` | MikroORM migration wrapper |
| `unified-strict-production-profile.js` | Strict reusable production profile |

The older direct-adapter examples remain for users who install adapter packages directly.
